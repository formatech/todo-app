# Todo App

A PhoneGap Todo App, using onsenui components + example for the contacts plugin

# Installation
Clone or download the app, and run
n
```
// run in browser
phonegap serve 

// or
phonegap run android

// or
phonegap run ios
```

# TypeScript 

```
// inside the project directory run
tsc --watch

// or
tsc --watch --project . 
```
